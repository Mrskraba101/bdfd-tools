Image Generator

?card

All params optional
Use %23 instead of # in URLs
Identity
username
Display name shown on the card. Default: "User"
user
Numeric Discord user ID shown as #id
avatar
HTTPS URL for the avatar image. Used as fallback if another source fails.
title
Small label beneath the name. e.g. Admin, Mod
bio
Short bio text. Max 120 chars, displays across 2 lines.
Appearance
theme
Card color theme. dark · cyberpunk · pastel · minimal (default: dark)
color
Hex accent color. Use %23 instead of # in the URL. Accepts #rgb or #rrggbb.
bg
URL of a custom background image.
overlay
Overlay on top of background. dark · light · none (default: dark)
blur
Background blur amount. 0–20 (default: 0)
glow
Enable glow effect around accent. true / false (default: false)
Layout
style
Card layout/size.
full — 620×200px (default)
compact — 420×130px
banner — 680×260px
Stats (optional)
level
Enables the XP progress bar when set.
xp
Current XP value for the progress bar.
xpmax
Max XP. If omitted, bar shows at ~65%.
rank
Shown as #n on the card.
messages
Message count, auto-formatted. 1200 → 1.2K
streak
Activity streak. e.g. 30d
Badges
badges
Comma-separated list, max 8.
vip, mod, booster, dev, staff, bot, early, verified, premium, partner
vip mod booster dev staff bot early verified premium partner
URL Structure
Base URL:
bdfd-tools.netlify.app/card

Example:
/card?username=Ash&level=5&xp=340&xpmax=500&badges=dev,verified&theme=dark&glow=true&color=%2300ffcc
