🎴 Discord Card API

Generate Discord-style profile cards as PNG images via URL parameters — drag & drop ready for Netlify.

Live Examples

API Reference

Endpoint

GET /card?[params]

All parameters (all optional)

username – display name (default: "User") user – user ID shown as #id avatar – https URL to avatar image title – short label tag bio – short description theme – dark | cyberpunk | pastel | minimal (default: dark) color – hex override for accent e.g. #ff0080 bg – https URL for background image overlay – dark | light | none (default: dark) blur – 0-20 background blur amount glow – true | false style – full | compact | banner (default: full) xp – current XP value xpmax – max XP for progress bar level – level number rank – rank number messages – message count streak – streak in days badges – comma-separated: vip,mod,booster,dev,staff,bot,early,verified,premium,partner
