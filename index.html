Image Generator

?card

All params optional
Use %23 instead of # in URLs
Identity
username
Display name shown on the card. Default: "User"
user
Numeric Discord user ID shown as #id
avatar
HTTPS URL for the avatar image.
title
Small label beneath the name. e.g. Admin, Mod
bio
Short bio text. Max 120 chars.
Appearance
theme
Card color theme. dark · cyberpunk · pastel · minimal
color
Hex accent. Use %23 instead of #.
bg
URL of a custom background image.
overlay
dark · light · none (default: dark)
blur
Background blur. 0–20
glow
Glow effect. true / false
Layout
style
full 620×200px · compact 420×130px · banner 680×260px
Stats
level
Enables the XP progress bar.
xp
Current XP value.
xpmax
Max XP for the progress bar.
rank
Shown as #n on the card.
messages
Auto-formatted. 1200 → 1.2K
streak
Activity streak. e.g. 30d
Badges
badges
Comma-separated, max 8. vip, mod, booster, dev, staff, bot, early, verified, premium, partner
vipmodboosterdevstaffbotearlyverifiedpremiumpartner
URL Structure
Base URL:
bdfd-tools.netlify.app/card

Example:
/card?username=Ash&level=5&xp=340&xpmax=500&badges=dev,verified&theme=dark&glow=true&color=%2300ffcc
